Project Title: Interactive Data Visualization Dashboard
Objective: To create an interactive dashboard that allows users to visualize and explore datasets dynamically using Plotly Dash and Matplotlib.
Tech Stack: Python, Pandas, Plotly (Dash), Matplotlib, VS Code
Dashboard Features:
Interactive dropdown filter by category
Plotly Line Chart for Sales Trends
Matplotlib Bar Chart for Profits
Dynamic updates when user changes input
Clean, dark-themed visualization
